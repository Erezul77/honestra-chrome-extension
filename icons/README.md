# Honestra Extension Icons

The icons/ folder should contain PNG files for the extension icon:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

## Creating Icons

You can create these icons using any image editor. The recommended design is:
- Background: Blue (#4A90E2)
- Icon: White shield emoji (üõ°Ô∏è) or similar guard/shield symbol
- Style: Clean, minimal, modern

### Quick Method using Online Tools:

1. Use https://www.favicon-generator.org/ or similar
2. Upload a simple shield icon or create one
3. Generate all sizes at once
4. Download and place in the icons/ folder

### Alternative: Use a simple colored square temporarily

For testing, you can create simple colored PNG files with any image editor.
The extension will work even with placeholder icons.
